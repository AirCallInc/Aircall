Partner Management


• From this section Admin can add the new partner, edit detail of previously added partner, Active/Inactive a partner and add Affiliates client in the partner.


• All the partners which are added by the admin will be listed in this section same as displayed in the below image.


• From the search field admin can search a partners with his/her name.


• If admin wants to inactive any partner then he/she can do it by following below steps:


1) Find the partner from the listing


2) Click on checkbox(left aligned) of that partner


3) Click ‘Inactive’ button


• Admin can search partners from search field.


• Admin can update partner from click on ‘edit’ button in the list.


• To add a new partner, Follow below steps :


1) Click on ‘Add Partner’ Button.


2) Admin will be asked Below details of partner.


3) Add the details of partner.


4) Click on ‘Add’ button.



       



    • Below are the fields to Add/Edit Partner :

Affiliate Clients :


• Admin can add and remove affiliate clients for the particular partner in partner add/ edit section as per show in below image.


• For add new affiliate admin will search and select the client from list of clients and click on ‘Add Affiliate’ button.


• Admin can remove the client from affiliate by click on ‘Remove’ button in row respective client.


• Admin also can view particular client detail by click on ‘View’ button in raw of respective client.
