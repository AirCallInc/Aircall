  1. Client Management

Client Profile


• From this section admin can add, edit, active, inactive and delete the client.


• Client who is inactivated or deleted by admin , cannot login into the application. Only activated client can login into the client application and client portal.




• From listing, User can also go to the units and see the client’s added units.


• The following are the details of client profile:


o The fields marked (*) are compulsory.



Field

Function

HVAC Unit (*):

A client will specify Unit’s age, while the sign up process which will help for marketing purpose.

Note: Any functionality will not be depended on this question’s answer it just be stored as information and for marketing purpose only.

First name (*):

Name of client.

Last name (*):

Last name of client.

Email Address (*):

Client’s E-mail id that can be used to sign into the client portal as well as the client application. This id will also be used by the admin to contact the client and to reset passwords.

A client will receive all the email sent by the Air call system to this registered email address.

When an admin will add a client or client will register with Aircall from website or application, the system will send a Sign up & Welcome email.

Password (*):

Password that is used to login into the client portal as well as the application.

Contact Numbers (*):

Contact details of client.

Below are the different contact numbers. One is compulsory.

  1. Mobile Number
  2. Office Number
  3. Home Number

Status:

Status of the client.

It should be of either Active or Inactive.

Affiliate Id:

Defines the partner code (if any).

When an admin will click on the “Affiliate Id”, the admin will be redirect to the related partner profile page, where all the clients who has registered with the same Affiliate Id will be displayed.

Admin can add the number of percentage for the commission from the partner management.

The admin will be able to see from Partner Report the amounts which, Aircall have to pay to the partner as Sales commission.

Client Addresses:

View of all client’s addresses.

Payment Methods:

View of all client’s payment methods.




• Email id and password added by the admin , using that user can log into the application as well as the client portal.


• The user can also register directly from the application and portal.


• Details entered during the sign up will be filled up automatically in the contact us form and in the checkout process.
