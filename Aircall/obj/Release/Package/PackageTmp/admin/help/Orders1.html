Orders


• From this section, Admin can add show listing of order and it’s detail which is added by employee from application or added by admin from admin portal.


• Admin can search order by Client name, Employee name or Date range from the top of section, Admin have to enter client name or employee name or date range in their respective field and click ‘Search’ button.


• If admin want to clear searched data and he/she want to show all data again then click ‘Clear’ button to show all datas.


• Admin can search in current displaying data from any column by just typing in search field.


• If the admin wants to delete any order then he/she can do it by following below steps:


1) Find the order from the listing which admin wants to delete

2) Click on checkbox(left aligned) of that partner

3) Click on red ‘Delete’ button



• Admin can view all the detail of any order by click on ‘View’ button from listing of orders. Details will be shown as per below image.




• Admin can add new order on behalf of client for purchase any part by click on ‘Add Order’ button in order section.


•When admin click on button admin will show screen like below image to fill-up all information to purchase part.



• Admin can make payment on behalf of client, there is 3 type of payment available in admin portal as shown in image below.


1) Check payment : User can make payment from check and admin will fill check information while place the order.

2) CC on File : When admin will select this type for payment then system will get data of card which is client selected as default card.

3) New CC : This option is use for use new card for payment rather then default card.



• Below are the fields to Add Orders :


Field

Functions

Items (*)

Enter item name which client need to purchase and click on search and select part from below list.

Qty (*)

Enter Quantity of selected part from above list and click on ‘Add Part’ button so part will be added in listing of parts as shown in above image. The quantity of any part can update and delete from listing.

Subtotal Amount (*)

This field just to show total amount of payment of all the parts. this field is not editable.

Client (*)

Enter name of client in field and click on ‘search’ so it will suggest name of client list, so from that select client.

Address(*)

Address of the partner.

State(*)

State of the partner.

City(*)

City of the employee.

Zip Code(*)

Zip Code of the employee.

Email(*)

Default email address of selected client will automatic fill-up there, admin can change it, after payment complete email will be send to this email address along with CC email address(if provided).

Charge By (*)

Method of client to charge.

This could be of below 3 types:

  1. Check (if picked, this option should expand to show fields to enter new cc information)
  2. CC on file
  3. Below New CC (if picked, this option should expand to show fields to enter new cc information)

Check number (*)

Check number of check given by client (Check Information).

Bank name (*)

Bank name of check given by client (Check Information).

Check Date (*)

Check date of check given by client (Check Information).
